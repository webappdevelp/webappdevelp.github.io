教程一 教了大家在Github上创建所需的仓库，教程二则将重点讲Hexo的使用。

Hexo是啥

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。所以，在这里你可能需要去了解Markdown、Pug或其他渲染引擎的语法。
关于Markdown的编辑，这里可以提供一个在线编辑器https://stackedit.io/app
